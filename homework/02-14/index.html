2-3 Questions a User Might Have

1. Which airports are the most frequestly-visited?

2. Which airlines provide the best on-time service from my local airport?

3. Which airports most frequently suffer delays?

Visualization Strategies

1. Map of airports geographically, with bubbles representing each airport and edges between the nodes representing the amount of direct connections between the two. A user can alter this map via filters and via hovering/clicking on nodes directly.

2. Bar charts showing the percentage of flights which are delayed by airline, within the applied filters.
